# Netflix-Viewership-Project
Power BI project analyzing Netflix content performance, audience engagement, and viewing trends
This project is an interactive Power BI dashboard that analyzes Netflix viewership trends and audience engagement patterns using data visualization and business intelligence techniques.

The dashboard explores:
- Content type performance (Movies vs Series)
- Growth in Netflix engagement over time
- Language-based viewing trends
- Seasonal release performance
- Top-performing content
- Series popularity by season

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization
- Data Cleaning & Transformation

## Key Insights
- Series generated significantly more viewing hours than movies
- English-language content dominated overall engagement
- Long-running series consistently attracted higher viewership
- Netflix engagement showed strong growth over recent years

## Dashboard Features
- Interactive slicers and filters
- Multi-page report structure
- KPI cards and analytical visuals
- Insight-driven storytelling
- Custom Netflix-themed dashboard design

## Project Objective
The goal of this project was to analyze viewing behavior across Netflix content and transform raw data into actionable insights through interactive visual analytics.
